## 快速开始

### 方式1：一键启动（推荐）

#### macOS/Linux
```bash
cd tsh_risk_app
chmod +x run_app.sh
./run_app.sh
```

#### Windows
```cmd
cd tsh_risk_app
run_app.bat
```

### 方式2：手动启动

```bash
# 进入目录
cd tsh_risk_app

# 创建虚拟环境（首次）
python -m venv venv

# 激活虚拟环境
# macOS/Linux:
source venv/bin/activate
# Windows:
# venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run app.py
```

### 方式3：使用conda

```bash
# 创建conda环境
conda create -n tsh_risk python=3.9

# 激活环境
conda activate tsh_risk

# 安装依赖
pip install -r requirements.txt

# 启动
streamlit run app.py
```

## 详细使用指南

### 第一步：输入基本信息

在左侧边栏输入以下必填项：

| 参数 | 建议值 | 说明 |
|------|--------|------|
| **TSH** | 1.45 | 您的甲状腺刺激素检测结果（mIU/L） |
| **年龄** | 45 | 患者实际年龄 |
| **性别** | 选择 | 生物学性别 |
| **BMI** | 27.5 | 体重(kg)/身高²(m²) |

**💡 提示**: TSH正常参考范围为 0.4-4.0 mIU/L

### 第二步：添加可选指标（推荐）

如有实验室检查结果，点击勾选框添加：

- **HDL**: 高密度脂蛋白胆固醇（正常>40 mg/dL）
- **TG**: 甘油三酯（正常<150 mg/dL）
- **CRP**: C反应蛋白（正常<3 mg/L）
- **HbA1c**: 糖化血红蛋白（正常<5.7%）

**💡 提示**: 添加更多指标可提高预测准确性

### 第三步：查看结果

#### 1. 风险指标卡片（页面顶部）

四个彩色卡片显示：
- **糖尿病风险**: 预测概率（百分比）
- **风险等级**: 低/中/高风险
- **OR值**: 相对于参考TSH的比值比
- **TSH百分位**: 在人群中的位置

#### 2. 风险解读（左侧）

根据风险等级显示不同颜色区域：
- 🟢 **低风险** (<5%): 继续保持
- 🟡 **中等风险** (5-15%): 需要关注
- 🔴 **高风险** (>15%): 建议就医

#### 3. TSH分析（左侧下方）

显示TSH的临床意义：
- TSH水平解读
- 人群百分位数
- 参考范围对比

#### 4. 风险仪表盘（右侧）

动态仪表盘直观显示：
- 当前风险位置
- 风险区域划分
- 与平均值比较

#### 5. 剂量反应曲线（页面中部）

交互式图表展示：
- TSH与糖尿病风险的关系曲线
- 您的位置标记（红色星标）
- 低/中/高风险区域

**💡 交互**: 鼠标悬停查看不同TSH值对应的风险

#### 6. 个性化建议（页面底部）

分为两部分：
- **基于TSH水平的建议**: 甲状腺相关
- **基于代谢风险的建议**: 糖尿病预防

## 常见问题

### Q1: 提示"ModuleNotFoundError: No module named 'streamlit'"

**解决**: 安装依赖包
```bash
pip install streamlit plotly numpy pandas
```

### Q2: 浏览器没有自动打开

**解决**: 手动访问 http://localhost:8501

### Q3: 端口被占用

**解决**: 修改端口
```bash
streamlit run app.py --server.port 8502
```

### Q4: TSH单位转换

本工具使用 **mIU/L**（国际单位）
- 如您的报告为 μIU/mL，数值相同，无需转换
- 如为 IU/L，需除以 1000

### Q5: 模型准确性

- 本模型基于NHANES美国人群数据
- 对中国人群的适用性可能有限
- 预测结果仅供参考，不能替代医生诊断

### Q6: 如何更新模型

编辑 `app.py` 文件中的 `MODEL_COEFFICIENTS` 字典，修改系数后重启应用。
